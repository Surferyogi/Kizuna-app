# Kizuna-app
Kizuna (Japanese for "deep bond") is a highly secure, privacy-focused messaging application designed for one-on-one or group communication, enabling people to connect, share information, and coordinate schedules. Kizuna symbolises connection that is not easily broken, one formed through shared experiences, love, care, trust and hardship. Kizuna carries warmth, loyalty and a sense of strength, bonding the invisible thread between our lives.
